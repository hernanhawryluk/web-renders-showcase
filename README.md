## Description

A web platform developed from scratch to showcase and promote high-quality architectural renders. This project was created as part of my own venture and stands out for its custom design, unique illustrations, and a wide range of features implemented with HTML, CSS, and JavaScript. Both the design and the code, along with all its content, were developed by myself.

## Características

- **Mobile First:** Designed and developed with a "Mobile First" approach to ensure an optimal experience on both mobile and desktop devices.
- **Adaptive Design:** Dynamically adjusts to various screen sizes, ensuring a seamless visual presentation at all times.
- **Dynamic Navigation Bar:** Adapts its color and shape based on the section and screen size, highlighting the user's current location for more intuitive navigation.
- **Slideshow:** Showcases cover renders through an engaging carousel that presents multiple works attractively.
- **Modal for Viewing Photos:** Allows users to expand images and explore every detail within an interactive modal.
- **Interactive Buttons:** Adds dynamism to the site with animated buttons that capture user attention and enable actions like adding new content with a simple click.
- **360 Renders:** Offers clients the opportunity to explore panoramic renders directly on the website for an immersive experience.
- **Automated Quoting:** Provides a real-time quoting system that automatically calculates prices based on the client's project size and type.
- **Real-Time USD Exchange Rate API:** Keeps prices updated for Argentina by querying an API to retrieve the daily USD exchange rate.
- **Data-Validation Form:** Ensures data accuracy by performing pre-submission checks, clearly indicating to the user where adjustments are needed if discrepancies are found.
- **Animations:** Page objects and elements enter the scene with smooth and elegant transitions.

## Tecnologías Utilizadas

- **HTML:** The robust structure of HTML enables a consistent presentation of the offered services.
- **CSS:** Custom styling in CSS ensures a unique and appealing appearance that adapts to screen width.
- **JavaScript:** Interactivity and advanced functionalities come to life through native JavaScript code, carefully tailored to provide a unique user experience.
